Packet Mirage is provided **for educational, defensive research, and authorized security testing purposes only**.  
It is designed to demonstrate concepts of traffic obfuscation, deception, network privacy, and WiFi security testing.

## Critical Legal Warning

⚠️ **UNAUTHORIZED USE IS ILLEGAL**

This project contains two distinct tools with different risk profiles:

### packet_mirage.py (Traffic Obfuscation)
- Generates fake network traffic for obfuscation and noise
- **Lower risk** - primarily for privacy demonstration
- Use requires network access only

### wifi_poison.py (WiFi Deauthentication Attack)
- Sends deauthentication frames to disconnect WiFi clients
- **HIGH RISK** - constitutes a network attack if used without authorization
- **Violations:**
  - **US:** Computer Fraud and Abuse Act (CFAA) - up to 10 years imprisonment
  - **EU:** Network and Information Systems Regulations (NIS2)
  - **UK:** Computer Misuse Act 1990
  - **Most countries:** Unauthorized network interference is a criminal offense
